By LimChaeJune
Posted 2022-05-07 Updated 2022-05-16 1 min read

NPM - Install Error Code 128

노트북으로 작업을 한 것을 푸쉬하고 데스크톱에서 pull을 받아 npm install을 실행했더니 아래와 같은 오류 메세지가 떴다.

구글링으로 찾아봤더니 npm 버전이 최신이 아니라 뜨는 거라고 나오더라 그래서 아래와 같이 실행하고 다시 npm install을 실행하니 잘 됐다!

npm install -g npm
